# モーニングコールLINEbot

AWS Lambdaで動作するモーニングコールLINEbotです。
このリポジトリではAWS Lambdaで使用するソースコード・ライブラリを管理しています。

## 使用技術

- AWS Lambda
- AWS EventBridge
- LINE Messaging API
- Node

## 環境変数

|  Key  | Value |
| --- | --- |
|  LINE_ACCESS_TOKEN   |  LINEbotのアクセストークン   |
|  LINE_USER_ID   |  LINEのユーザID   |
